## Flagship fallback tokens

If a fetch fails, these compact summaries let you proceed without the full page.

### Mercury (dark fintech)
- **Backgrounds:** Deep Space `#171721` (outer), Midnight Slate `#1e1e2a` (sections), Graphite `#272735` (interactive surfaces)
- **Text:** Starlight `#ededf3` (primary), Silver `#c3c3cc` (secondary)
- **Accent:** Mercury Blue `#5266eb` — **primary CTAs only**, never for text/decoration
- **Borders/dividers:** Lead `#70707d`
- **Type:** light-weight sans for headlines (fallbacks: Inter, Manrope); display ~65px, heading ~32–49px, body 16px, line-height 1.1–1.5; subtle positive letter-spacing
- **Shape:** base unit 4px; spacious; max-width 1200px; section gap 80–120px; **pill buttons** (radius 32–40px), inputs radius 32px, cards radius 0
- **Rules:** no shadows for elevation (use color/opacity shifts); no heavy font weights; high contrast; generous breathing room
